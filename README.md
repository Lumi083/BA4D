# BASpark 4 Droid (BA4D)

## 使用

1. 下载安装BA4D
2. 下载安装配置 Shizuku
3. 启动

## 运行源码

1. 用 Android Studio 打开本目录
2. 等待 Gradle Sync 完成
3. 连接 Android 设备（建议 Android 8+）
4. 安装并启动应用
